Project Description: 
* Walmart Bloons Tower Defense 
* In each wave, a group of balloons spawn on a path and they try to get to the end of the path. If they do, you lose some health. In order to stop them from getting there, you place some turrets down to destroy the balloons and prevent them from reaching the end point. Each time the game is run, a new randomly generated map is created.


How to Run:
* Things to install:
   * Python
   * PIL
   * Requests
* Open the folder and run the file called ‘main’


Commands:
* ‘Escape’ pauses the game and brings you to the pause screen
* ‘p’ brings you to the game
* ‘h’ brings you to the help screen
